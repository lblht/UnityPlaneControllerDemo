# **Unity Plane Controller Demo**

_A simple demo project for a plane controller in Unity._

## **Controls**

| Key     | Action       |
| ------- | ------------ |
| **X**   | Start Engine |
| **W/S** | Pitch        |
| **A/D** | Yaw          |
| **Q/E** | Roll         |
